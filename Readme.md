# 📄 Portfolio — Moustapha FAYE

> CV interactif et responsive développé en HTML, CSS et JavaScript vanilla.

---

## 📌 Présentation

Ce projet est un **portfolio personnel** sous forme de CV en ligne.  
Il présente mon parcours académique, mes compétences, mes expériences professionnelles ainsi qu'un formulaire de contact.  
Le design est entièrement responsive et s'adapte aux écrans desktop, tablette et mobile.
Inspire d'un cv professionel telecharger sur canvas

## ✨ Fonctionnalités

| Fonctionnalité | Description |
|---|---|
| 📱 **Responsive design** | Adapté desktop, tablette et mobile |
| 🍔 **Menu hamburger** | Navigation en colonne sur mobile avec icônes |
| 🌙 **Mode sombre** | Bascule jour/nuit via un bouton fixe |
| ⬆️ **Retour en haut** | Bouton flottant qui apparaît au défilement |
| 📊 **Barres de compétences** | Niveau de maîtrise affiché visuellement |
| 📬 **Formulaire de contact** | Validation JS côté client (simulation d'envoi) |
| 🔗 **Liens projets** | Accès direct aux projets déployés |

---

## 🛠️ Technologies utilisées

- **HTML5** — Structure sémantique
- **CSS3** — Mise en page (Grid, Flexbox), variables CSS, media queries
- **JavaScript (ES6)** — Interactions dynamiques (hamburger, dark mode, scroll, formulaire)
- **[Font Awesome 6](https://fontawesome.com/)** — Icônes (CDN)

> ⚠️ Aucun framework ni bibliothèque JS — 100 % vanilla.

## 📱 Comportement responsive

| Écran | Comportement de la navigation |
|---|---|
| **Desktop** (> 768px) | Menu horizontal avec icônes et texte |
| **Tablette** (≤ 1024px) | Menu horizontal compact |
| **Mobile** (≤ 768px) | Bouton hamburger → menu vertical en colonne |
| **Très petit** (≤ 400px) | Mise en page simplifiée, textes réduits |

---

## 📬 Formulaire de contact

Le formulaire valide les champs côté client (nom, email, message) stocke avec LocalStorage et affiche un message de confirmation.  
**Aucun email n'est réellement envoyé** — il s'agit d'une simulation JavaScript.
